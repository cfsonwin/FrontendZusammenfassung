  • Introduction

    1. AJAX = Asynchronous JavaScript And XML.
    2. AJAX is not a programming language. AJAX just uses a combination of: A browser built-in XMLHttpRequest object (to request data from a web server) JavaScript and HTML DOM (to display or use the data)
    3. Application
      • Update a web page without reloading the page
      • Request data from a server - after the page has loaded
      • Receive data from a server - after the page has loaded
      • Send data to a server - in the background
      • According to Event update UI without reloading
      • 鼠标移入显示菜单，用则加载，不用就不会加载
      • 快滚动到底部时添加新的内容
      • 图形交互界面
    4. Working flow
    5. For security reasons, modern browsers do not allow access across domains. This means that both the web page and the XML file it tries to load, must be located on the same server.
    6. Across domin strategy
      • Same-origin Policy: 网页url和ajax请求的url必须协议、域名、端口一致
      • 解决方案1: jsonp：使用script标签作为跨域的媒介，服务器端返回的数据为js代码
