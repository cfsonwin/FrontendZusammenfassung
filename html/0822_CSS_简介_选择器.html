    6. Comments: /*----------CSS Comments-----------*/

  • CSS Selectors

    • Universal selector:“ * ”
    • CSS element Selector, for example <p>

    • CSS id Selector: hash (#) character. [#id {************} ]
    • CSS class Selector: [.className {************}] HTML elements can also refer to more than one class.
    • CSS Grouping Selector: selects all the HTML elements with the same style definitions. like [ele1, #id, .class, ... {************} ]
    • CSS Combinators [Go example]:
      • child selector:father_ele > child_ele {************}
      • Descendant Selector: [space]. All elements that are descendants of a specified element.
      • Adjacent Sibling Selector : ele1 + ele2 {************}
      • General Sibling Selector: ele1 ~ ele2 {************}

    • CSS [attribute] Selector:
      • element[attribute="value"]{************} select elements with a specified attribute and value.
      • element[attribute]{************} Selects all elements with a target attribute
      • element[attribute^=value]{************} attribute value begins with value
      • element[attribute$=value]{************} attribute value ends with value
      • element[attribute*=value]{************} attribute value contains value
    • Pseudo-class Selector:[Go example]
      • Pseudo-classes:A pseudo-class is used to define a special state of an element. For example, it can be used to: Style an element when a user mouses over it...
      • The [:first-child] pseudo-class matches a specified element that is the first child of another element;
        :last-child;
        :nth-child(),value can be "odd/2n+1" "even/2n" "n"
      • :first-of-type/:last-of-type/:nth-of-type()
      • The [:lang] pseudo-class allows you to define special rules for different languages.
      • [:not()] element that is not a "xx" element
      • Links can be displayed in different ways [in order]:
        :link(unvisited link) -> :visited(visited link) -> :hover(mouse over link) -> :active(selected link)
        • pseudo class for input element

    • Pseudo-element Selector:
      • to style specified parts of an element, use "::".
      • [::first-letter] Style the first letter of an element;
      • [::first-line] Style the first line of an element;
      • [::selection] matches the portion of an element that is selected by a user.
      • The [::marker] pseudo-element selects the markers of list items.
      • [::before] insert some content before the content of an element; [::after]insert some content after the content of an element.

  • Cascading Order:

    • inline style > id Selector > (pseudo-)class Selector > element Selector
    • Combinators adding order together [like 1000,100,10,1 each type], but not have the priority than next level
    • Grouping Selector don't add
    • use the style defined later with same priority
    • with "!importent" get the highest priority.
