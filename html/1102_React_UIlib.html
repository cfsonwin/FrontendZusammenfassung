  • 修改默认配置文件

    1. 终端中输入 yarn eject 并确认
    2. antd样式按需引入
      • yarn add @craco/craco
      • /* package.json */
        "scripts": {
        - "start": "react-scripts start",
        - "build": "react-scripts build",
        - "test": "react-scripts test",
        + "start": "craco start",
        + "build": "craco build",
        + "test": "craco test",
        }
      • Then create a craco.config.js at root directory
    3. 修改antd样式
      • /* src/App.js */
        - import './App.css';
        + import './App.less';
      • /* src/App.less */
        - @import '~antd/dist/antd.css';
        + @import '~antd/dist/antd.less';
      • yarn add craco-less
      • modify craco.config.js like below:
        const CracoLessPlugin = require('craco-less');
        module.exports = {
        plugins: [
        {
        plugin: CracoLessPlugin,
        options: {
        lessLoaderOptions: {
        lessOptions: {
        modifyVars: { '@primary-color': '#1DA57A' },
        javascriptEnabled: true,
        },
        },
        },
        },
        ],
        };
      • By adding modifyVars option of less-loader here, we should see a green button rendered on the page after rebooting the server now.
