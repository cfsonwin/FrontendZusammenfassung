  • 单位

    1. px：像素
    2. 100%：百分比，设置相对于父元素的百分比
    3. em：相对于font-size的大小，默认1em为16px
    4. rem：相对于根【root】元素的字体大小

  • 颜色单位

    1. 可以用颜色名设置各种颜色
    2. 使用rgb、rgba值，0~255（0%~100%）
    3. 语法：rgb(xx, xx, xx)；rgba(xx, xx, xx, 0-1的透明度)
    4. 十六机制rgb表示：【# xx xx xx】每个xx是00-ff
    5. HSL、HSLA值，h表示色相，s是饱和度，l是亮度
    6. h:0-360; s:0%-100%; l:0%-100%

  • 文档流 normal flow

    1. 网页时一个多层的结构，最底下的一层是文档流
    2. 创建的元素默认在文档流中，所以元素有两个状态,在文档流中或者不在文档流中。
    3. 块儿元素在文档流中独占一行，默认宽度是父元素的宽度。默认高度被子元素撑开
    4. 行内元素不换行，自左向右水平排列，自动换行。默认宽度和高度都取决于内容。

  • 盒模型 box model

    1. 页面中的所有元素都是一个矩形的box
    2. 页面的布局就是把每个box摆放在不同的位置
    3. 组成部分：content border padding margin
