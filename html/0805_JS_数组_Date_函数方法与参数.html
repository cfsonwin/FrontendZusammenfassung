  • 数组 Array

    1. 数组也是一个对象, 用来存储数据
    2. 由索引index和值value组成
    3. 创建新数组var arr = new Array();或者 var arr = [];
    4. 添加元素: arr[0] = 10;
    5. .length(属性, 不是方法) 返回数组长度, 可以用来修改数组长度: 删除多余元素或创建空元素
    6. 可以储存任意类型, 包括对象以及函数
    7. -1倒数第一个元素
    8. 方法:
      • 转为字符串
        1. .join(“连接符”) 将原数组转化为字符串, 默认用逗号连接。specify the separator
        2. .toString()方法也可以转为字符串
      • 操作第一个或者最后一个数据
        1. .push() 在数组后面添加一个或多个元素并返回数组长度
        2. .pop() 删除数组的最后一个元素, 并返回删除的元素
        3. .unshift() 在数组开始添加一个或多个元素并返回数组长度
        4. .shift() 删除数组的第一个元素, 并返回删除的元素, 相当于向左平移
      • 使用index number操控任意位置数据, 包括修改数据以及添加数据。直接delete array[0]不会删除数据, 而会变成undefined
      • Merging (Concatenating) Arrays
        1. .concat(arr1, arr2) 合并数组到后面并返回, 原数组不变
      • Splicing [删除+添加元素] and Slicing Arrays[切片]
        1. .slice(start_position, end_position) 切片, 不会改变原数组, 而是返回一个新数组
        2. .splice(start_position, count, new_element...) 从start_index开始删除count个, 并添加new_elememt进入数组, 改变原数组并返回删除的元素.
      • .reverse() 反转数组, 直接改变原数组
      • 排序
        1. .sort() 排序, 默认按Unicode编码排序, 直接改变原数组, 可以传入一个回调函数。
        2. points.sort(function(a, b){return b - a});
        3. 将数组中的元素作为实参调用回调函数, 且a一定在b前面。返回大于0的值则会交换。
        4. Math.max.apply(null, arr); //find the highest number in an array
    9. 遍历:
      • for(var i=0; i < arr.length; i++){ console.log(arr[i]); }
      • .forEach(function(value, index, array){});方法: calls a function (a callback function) once for each array element.
      • Note that the function takes 3 arguments: The item value; The item index; The array itself;
      • .map()
        1. 不会改变原数组，与forEach()相反
        2. the function takes 3 arguments: The item value The item index The array itself When a callback function uses only the value parameter, the index and array parameters can be omitted
      • .reduce(Func, initial value of total)
        1. 不会改变原数组
        2. 遍历后产生一个single value
        3. function takes 4 arguments: The total (the initial value / previously returned value); The item value; The item index; The array itself
      • The reduceRight() works from right-to-left in the array.
      • .every()
        1. The every() method checks if all array values pass a test.
        2. Note that the function takes 3 arguments: The item value; The item index; The array itself;
      • .some(): The some() method checks if some array values pass a test.
      • .find(): returns the value of the first array element that passes a test function.
      • .filter(): Creates a new array with every element in an array that pass a test
      • .indexOf() 查找元素角标, 没有找到的话返回-1, 有的话返回找到的第一个

  • 时间 Date

    1. var d = new Date(); 封装执行代码时的时间
    2. Date(mm/dd/yyyy hh:mm:ss)
    3. 方法:
      • .getDate() 当前日期几号
      • .getDay() 当前日期周几, 返回0~6, 周日是0
      • .getMonth() .getFullYear() .getYear()... 当前日期月份
      • .getTime() 获取当前时间的时间戳

  • 函数的方法 .call() .apply()

    1. 都会调用函数, 可以传入对象当参数, 传入的对象作为函数中的this替代window
    2. 通过函数对象来调用.已有function func(){};
      • func.call() 传入的对象后传实参
      • func.apply() 实参要封装到数组中传入

  • arguments:隐藏函数传入参数

    1. 类数组对象, 保存传入的实参
    2. .length属性
    3. .callee属性, 指代当前调用的函数函数对象
    4. 通过索引取值: arguments[]

  • 数组补充

    1. It is a common practice to declare arrays with the const keyword.
    2. It does NOT define a constant array. It defines a constant reference to an array. Because of this, we can still change the elements of a constant array.
    3. JavaScript has a built-in array constructor new Array(). But you can safely use [] instead.
    4. Array.isArray() to Recognize an Array
    5. .sort() .forEach() .map() 可以传入回调函数
