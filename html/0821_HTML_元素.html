0821 HTML元素

Back Homepage

  • 文档元素

    1. 多个空格会被解析为一个空格
    2. 使用html的 entities 表示特殊符号, 空格 大于号>
    3. meta标签: charset; name指定数据名称 content指定数据内容: 有keywords、description、

    <!Doctype html lang='en'> //文档声明, 使用HTML5版本的文件

    <html> //根标签

    <head>

    <meta charset='utf-8'> //设置元数据: 指明字符集

    <meta name='keywords' content='html学习'> //指明用于搜索的关键字

    <title>网页标题</title>

    </head>

    <body>

    </body>

    </html>

  • 标签

    1. 标题: h1~h6,一般用1-3.是一个块儿元素[block element],独占一行。 <hgroup></hgroup>标题组
    2. 段落: p。也是块儿元素
    3. 加重: em、strong。inline标签。行内元素或者内联元素
    4. 引用: blockquote--块儿元素; q--行内元素
    5. 换行: br; 分割线: hr
    6. 其他块儿元素: header/main/footer; 导航 nav; 主体相关内容: aside; 文章 article; 独立的区块 section
    7. 最常用: div; 对应的行内元素span

  • Block element and inline element

    1. 一般来说块元素用来布局, 行内元素用来展示文字
    2. 块儿元素什么都可以放, p元素内不可以放块儿元素
    3. 布局标签: 显示效果一样, 需要额外加css
      • header 网页的头部
      • main 主体部分
      • footer 网页的底部
      • nav 网页中的导航
      • aside 侧边栏
      • article 表现为独立的文章
      • section 表示一个独立的区块儿, 主要用div
      • 行内标签使用span

  • List

    1. ordered list; unordered list; description list.
    2. li: list item
    3. 第一列表dl, 使用dt表示定义内容, dd进行解释说明

  • 超链接a标签

    1. 行内标签, 可以放块儿元素, 可以用br换行
    2. 使用href属性指明跳转链接
    3. 使用target指定超链接打开位置, _self或者_blank, 是否新建标签打开标签,在内联框架中打开需要使用iframe的name属性
    4. 跳转到页面的其他位置:
      • 回到顶部: Top
      • 回到底部: Bottom
      • 去Entity部分笔记: Go Entity
    5. 开发中井号#可以用作占位符
    6. 使用javascript:;也可以用作占位符

  • 图像标签

    1. 使用img标签, 是自结束标签
    2. src属性添加路径
    3. alt属性会在图片不能加载的时候替换图片, 也是搜索引擎的关键字
    4. width heigth限制长宽, 如果只修改一个另一个会等比例缩小
    5. 替换元素, 不独占一行, 既有行内元素也有行内元素的性质
    6. jpg格式显示照片, 不支持透明; png支持复杂透明; gif支持动图以及透明
    7. webp是谷歌搞的格式, 小且功能强大, 兼容性不好
    8. 将图片使用base64进行编码, 将图片转换为字符

  • 音频视频文件

    1. audio是音频标签, src属性确定源
    2. 实际使用中会自己写css、js确定样式和功能
    3. 其他属性
      • controls: 属性添加控制界面
      • autoplay: 设置是否自动播放, 但是大部分网页不支持
      • loop: 是否循环播放
    4. video标签引入视频文件, 使用方式和audio差不多
