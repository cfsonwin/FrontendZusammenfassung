  • border

    1. width: 1-4值，上右下左
    2. color: 1-4值，上右下左
    3. style： solid dotted dashed double
    4. border会影响可见区大小，outline既不影响可见区大小，也不影响页面布局
    5. box-shadow设置阴影，四个参数：水平偏移量 垂直偏移量 模糊半径 颜色
    6. border-radius设置圆角
    例如：border: 2px red solid;
    例如：border-width: 2px 4px 6px 8px; border-style: solid; border-color: aliceblue rebeccapurple darkblue firebrick;
    加个圆角
    使用outline
    加个阴影

  • padding

    1. width和height只是内容区的大小，不包括padding border和margin
    2. 背景颜色会延伸到padding区
    3. 子元素设置100%只会铺满内容区

  • margin

    1. 同样四个方向，影响box的位置
    2. 上左margin会影响自身位置，下右margin影响其他元素
    3. 可以为负值
    4. 外边距折叠：垂直方向外边距会重叠
      • 兄弟元素之间都为正值取大值一正一负取和，两个负值取绝对值大的
      • 父子元素之间，相邻外边距子元素会传递给父元素
    父元素没有设置margin，也会和子元素一起向下移
    第一个margin-bottom：20 第二个margin-top：-10，最后间距为10px
    第一个margin-bottom：-20 第二个margin-top：-10，最后间距为-20px
    父元素使用clearfix，子元素外边距不会传递给父元素了

  • 水平布局

    1. marLeft+borLeft+padLeft+width+padRight+borRight+marRight = 父元素内容去宽度
    2. margin-right默认为auto自动调整大小
    3. margin-left和width也可以设置为auto
    4. 水平居中：设置width固定值并margin：0 auto；
      补充
    1. 文本居中：text-align: center;
    2. 居中图像，将左右外边距设置为 auto，并将其设置为块元素
    3. 左右对齐：position: absolute;right: 0px; / float: right;

  • 垂直布局

    1. 默认父元素高度被内容撑开
    2. 如果父元素设置了height子元素超出则会直接溢出，可用overflow解决
    3. overflow：wisible默认；hidden隐藏超出部分；scroll添加滚动条；auto自动
      补充
    1. 如果一个元素比包含它的元素高，并且它是浮动的，它将溢出其容器。可以使用overflow: auto;
    2. 单行文本垂直对齐：设置inline-height=height / 使用 position 和 transform 属性position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);

  • inline的box model

    1. 无法设置height和width改变大小
    2. 可以设置margin/border/padding，但不会影响垂直布局
    3. 水平方向的margin不会折叠
    设置了内外边距的行内元素 设置了内外边距的行内元素2

  • display

    1. inline:设置为行内元素
    2. block：设置为块元素
    3. inline-block：与 display: inline 相比，允许在元素上设置宽度和高度，并保留上下外边距/内边距。
      与display: block相比，inline-block在元素之后不添加换行符
    4. table：表格
    5. none：不在页面中显示，区别于visibility：hidden，不保留位置

  • 其他

    1. 清除list样式：list-style：none；
    2. 去掉a的下划线：text-decoration：none
    3. 设置box-sizing：border-box后，高度宽度为可见区大小
