  • Introduction

    1. ECMAScript 2015 was the second major revision to JavaScript. ECMAScript 2015 is also known as ES6 and ECMAScript 6.

  • Let

    1. To declare a variable
    2. Variables defined with "let" cannot be Redeclared (but var can be redeclared).
    3. Variables defined with "let" must be Declared before use. Using a let variable before it is declared will result in a ReferenceError.
    4. Variables defined with "var" are hoisted to the top and can be initialized at any time. Meaning: You can use the variable before it is declared.
    5. Variables defined with "let" have Block Scope. Variables declared inside a { } block cannot be accessed from outside the block. Before ES6 (2015), JavaScript had only Global Scope and Function Scope. Variables declared with the "var" keyword can NOT have block scope.
    6. Block Scope for example after if, else, while Loop and so on
    let x = "John Doe";
    let x = 0;
    // ERROR-> SyntaxError: 'x' has already been declared
    {
    let x = 2;
    }
    // x can NOT be used here

  • Const

    1. To declare a constant variable,and must be assigned a value when they are declared
    2. It does not define a constant value. It defines a constant reference to a value.
    3. Elements of constant array and properties of constant object can be changed
    4. Variables defined with "const" cannot be Redeclared.
    5. Variables defined with "const" cannot be Reassigned.
    6. Variables defined with "const" have Block Scope.
    7. Variables defined with "const" are also hoisted to the top, but not initialized (like var).
    const PI = 3.141592653589793;
    PI = 3.14; // ERROR
    PI = PI + 10; // ERROR

    // You can create a constant array:
    const cars = ["Saab", "Volvo", "BMW"];
    // You can change an element:
    cars[0] = "Toyota";
    // You can add an element:
    cars.push("Audi");
    const cars = ["Saab", "Volvo", "BMW"];
    cars = ["Toyota", "Volvo", "Audi"]; // ERROR

    // You can create a const object:
    const car = {type:"Fiat", model:"500", color:"white"};
    // You can change a property:
    car.color = "red";
    // You can add a property:
    car.owner = "Johnson";
    const car = {type:"Fiat", model:"500", color:"white"};
    car = {type:"Volvo", model:"EX60", color:"red"}; // ERROR
