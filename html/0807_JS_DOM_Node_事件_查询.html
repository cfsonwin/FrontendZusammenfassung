0807 Node 事件 查询

Back Homepage

  • 节点Node

    1. 文档节点：整个HTML文档，document,nodeType=9，value=null
    2. 元素节点：标签，标签名，nodeType=1，value=null
    3. 属性节点：元素属性，属性名，nodeType=2，value=属性值
    4. 文本节点：标签中的文本内容，text，nodeType=3，value=文本内容

  • 事件：

    1. 用户和浏览器的交互：点击按钮，鼠标移动，关闭窗口等。
    2. 属性：.onclick .onmousemove .ondblclick
    3. 在文档开头的js，要使用window.onload = function(){...};

  • 查询：

    1. 获取元素节点
      • .getElementById()
      • .getElementsByTagName()
      • .getElementsByName()
    2. 获取元素节点的子节点
      • .getElementsByTagName()
      • .childNodes
      • .firstChild
      • .lastChild
