0801 对象 数据类型

Back Homepage

  • 对象

    1. 内建对象: Math String Number等
    2. 宿主对象: BOM DOM 如document console等
    3. 自建对象: var obj = new Object(); new关键字用来创建对象
      • 添加属性: obj.Attribute = xxx;
      • 修改属性: obj.Attribute = new_xxx;
      • 删除属性: delete obj.Attribute;
      • obj["属性名"] = 属性值; 这种方法更灵活
      • 检查对象中是否有一个属性: "属性名" in obj;-->返回布尔值

  • 数据类型

    1. 基本数据类型: 存在栈区, 存储每一个变量与对应的值
    2. 引用对象类型: 栈区存储对象名以及堆区地址, 属性存在堆区
      1. obj2 = obj1; 通过obj2改属性的值也会导致obj1中属性值的改变, 改的是堆区里的值
      2. 将obj2 = null; 改变栈区中obj2的内存指向
      3. 比较两个引用对象, 则会比较地址

  • 对象字面量

    1. var obj = {属性名: 属性值, ... }; 与new方法是一样的, 最后一个属性后面没有逗号
    2. 这种对象不是一个构造函数, 不能使用new进行实例化
    3. 它是引用类型, 也就意味着对象名是一个指针, 当你把对象名赋值给另一个变量时, 你在新变量上所做的任何操作都会影响源对象
