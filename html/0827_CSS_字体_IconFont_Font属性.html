0827 字体 IconFont Font属性

Back Homepage

  • 字体

    1. color
    2. font-size[em一个字体大小]
    3. font-family:serif sans-serif monospace fantasy
    4. 如果把字体放在服务器端，css中加入@font-face{fong-family: fontName; src: url('...')}
    5. 注意版权问题

  • icon font

    1. 以类似于字体的格式插入网页的图标
    2. 可以调整大小和颜色
    3. font awesome
      • 下载解压后将css和webfont移动到项目中
      • 将css目录下的all.css引入网页中
      • 通过class引用字体
      • 也可以在css样式中使用：content=“字体的编码” 使用【fab】font-family: "font awesome 6 free" / 【fas】额外加入font-weight：900设置样式

      4. 例如在标签中添加class="fas fa-bell" style="color: red; font-size: 2em;"

      设置css样式li::before{font: var(...); content: '\xxx' }

      • line1
      • line2
      • line3
    4. 阿里的iconfont

  • line height

    1. 指的是每行字占的实际高度，通过line-height设置
    2. 字体框：font-size设置其高度
    3. font设置字体的所有属性：【font-weight font-style等】 字体大小/行高 字体族
    4. font-weight：normal bold（加粗） 100-900
    5. font-style：italic
    6. text-align：left right center justify
    7. vertical-algin：baseline基线对齐；top/bottom/middel对齐父元素顶部/底部/中间【不是严格意义上的居中】；
    8. 图片自身放到块儿中类似于文字也有一个基线，可以设置vertical-align取消基线和父元素之间的空隙
    9. text-decoration：none underline line-through overline 还可以指定颜色和样式【类似于border】
    10. 文本框设置white-space：normal nowrap【不自动换行】pre【保留空格和换行】
    11. 文本显示有限内容以及省略号：white-space：nowrap；overflow：hidden；text-overflow：ellipsis
