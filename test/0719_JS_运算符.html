0719 Operators

Back Homepage

  • 一元运算符

    1. + 正; 进行隐式数字类型转换
    2. - 负; 对非number的值，则先转化为number
    3. ++ 自增, a++=原值，之后a自增; ++a=自增后的值
    4. -- 自减.

  • 逻辑运算符

    1. ! 非，对布尔值取反, 任意数据类型两次取反将其转换为布尔值;
    2. && 与; 第一个为false则后面的不会检查
    3. || 或; 第一个为true则后面的不会检查
    4. 对非布尔值进行运算，会先转换为布尔值，再运算，并返回原值。

  • 赋值运算符

    1. = 右侧值赋给左侧变量
    2. += a+=5 == a=a+5
    3. -=
    4. *=
    5. /=
    6. %=

  • 关系运算符: 返回布尔值

    1. 对非数值，转换为数值在进行比较。布尔值视为0或1；任何值和NaN比较都为false
    2. 如果两边都是字符串, 则会将其转化为Unicode编码进行比较, 依次按位比较(英文排序)。所以比较两个数字字符串时，一定要转型。
    3. UTF-8编码在网页中用&#，但要跟十进制。
    4. 类型：
      • >
      • >=
      • <
      • <=
      • == 注意：NaN不和任何值相等，用isNaN()函数判断是否为NaN；undefined == null --> true；
      • !=
      • === 类似于==，但类型不同直接返回false; undefined === null --> false；
      • !== 类似于!=，但类型不同直接返回true

  • 分割运算符

    1. 同时声明多个变量，如var a=1, b, c;

  • 三元运算符: 条件表达式？语句一：语句二；

    1. 条件结果为true，执行语句一
    2. 条件结果为false，执行语句二

  • 运算符优先级

    先乘除后加减，相同优先级从左向右
